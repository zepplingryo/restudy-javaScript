# 重学 javaScript

## how to study?

首先 js 是一门上手 3 天 精通 30 年的语言

例如 js 三大核心 在我们实际开发中这些都是我们不可或缺的一部分

- 函数、闭包
- 异步
- 面向对象

同时 js 是一门脚本语言 同时仰仗着 chromium 在各大平台进行寄生(Atwood 定律 虽然 rust 已经开始帮助前端生态) 但是还是不影响 js 在传统 web 端有绝对的统治地位 同时在例如移动端、服务端、桌面端都有身影

所以我认为 js 基础是我们实现业务的基石
